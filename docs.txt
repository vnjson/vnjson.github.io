```yaml
# p - это сокращенное обозначение персонажа [Профессор]
# a - это сокращенное обозначение персонажа [Алиса]
- p: Привет мир
- a: Привет!
  key: мир!

```

Каждая строчка начинающиеся со знака ( - )(тире) является новым состоянием. Тоесть когда вы жмете кнопку что бы ваша новелла пошла вперед. То смениться состояние

## Если вы мало-мальски знаете javascript
```js
[
  {"p": "Привет мир"},
  {"a": "Привет"}
]
```
Каждое ключевое имя (-) начинающиеся с тире является излучаемым событием, которое вызывается когда выводится каждая строка
События можно отлавливать простейшим плагином
```js
vnjs.on('key', function (value) {
    alert('Привет' + value);
});
```
