Спецификация

Сцены написанные на YAML транслируются в json
Для того что бы разгрузить синтаксис

Каждое ключевое слово является событием

    
- jump: scene.label    

- menu:
   ?: Вопрос
   label-1: Вариант 1 
   lab-2: Вариант 2

Структура сцены

    
scenes
├───scene-1
│   ├───assets
│   |   ├───home2.jpg
│   |   ├───Nora.png
│   |   ├───school.jpg
│   │   └───main-theme.mp3
│   ├───start.yaml
│   ├───home.yaml
│   └───characters.yaml   
├───scene-2
│   ├───assets
│   |   ├───Nora.png
│   |   ├───Alice.png
│   │   └───song1.mp3
│   ├───chapter-1.yaml
│   ├───chapter-2.yaml
│   └───characters.yaml

Сцена в готовом виде

    
{
  "assets": [
    {
      "name": "home2",
      "url": "./scenes/assets/home2.jpg"
    },
    {
      "name": "main-theme",
      "url": "./scenes/assets/main-theme.mp3"
    }
  ],
  "characters": [
    { "name": "nr", "text": "Нора" }
  ],
  "home": [
    {"nr": "Some reply", "audio": "main-theme"},
    {}, {}, {}
  ],
  "start": [  ]
}

Объявление персонажей

    
#characters.yaml

- name: nr
  text: Нора

Как выглядит код label’a

- print: События о который пойдет речь, начались задолго до твоего рождения.
  audio: 
    name: main-theme
    action: play
    loop: true
    volume: 30
    speed: 0.3
  scene: school
